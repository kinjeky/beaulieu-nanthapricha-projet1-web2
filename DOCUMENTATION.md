## Choix technique
- Mettre end place un système BEM pour pouvoir créer un programme durable en suivant la logique que Figma donne dans le code css.
- Mettre un * devant chaque element dans la documentation du css pour permettre une meilleure lisibilité.
- Essayer de faire toute le code en premier pour ensuite le passer au peigne fin pour corriger toute faute.
- Faire un root à la fin pour voir quel sont les données les plus récurentees dans le code css donné par Figma.
